# Minecraft item sprite sources

These local PNGs are unmodified Minecraft inventory sprites retrieved from the
Minecraft Wiki/Fandom file API on 2026-07-16. Aestron bundles them so commands do
not hotlink the wiki or perform web requests during interactions.

- Armor sprites: [Minecraft Wiki item icon category](https://minecraft.wiki/w/Category:Item_icons)
- Sword sprites: [Minecraft Wiki item icon category](https://minecraft.wiki/w/Category:Item_icons)
- Golden apple: [Minecraft Wiki Golden Apple](https://minecraft.wiki/w/Golden_Apple)
